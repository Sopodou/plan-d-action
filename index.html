Service d'envoi — EmailJS

EmailJS envoie des e-mails directement depuis le navigateur, sans serveur. Créez un compte gratuit sur emailjs.com, configurez un service (Gmail, Outlook, SMTP…) et un template avec les variables to_email, subject, message. Puis décommentez le script EmailJS dans index.html.

Les paramètres sont sauvegardés automatiquement dans ce navigateur.
